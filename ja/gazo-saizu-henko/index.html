無料でオンラインで画像サイズを変更する方法

写真のサイズ変更は、ウェブサイトの最適化、メール送信、またはデジタルプラットフォームの要件を満たすために不可欠です。以下の手順に従ってください：

  1. 写真をアップロード：JPG、PNG、または WebP の画像を上部のドラッグゾーンにドロップするか、クリックしてファイルを選択します。
  2. サイズ変更モードを選択：縮小率（50%など）でサイズ変更するか、幅と高さのピクセル数を指定してサイズ変更するかを選択します。
  3. 縦横比を維持：画像が引き伸ばされたり歪んだりするのを防ぐため、「縦横比を維持」のチェックボックスをオンにしたままにすることをお勧めします。
  4. 処理してダウンロード：「一括サイズ変更」をクリックします。ローカル処理が完了したら、「画像を保存」または「すべての画像をダウンロード」を押して一括保存します。

よくある質問 (FAQ)

このツールを使用するとファイルサイズ（容量）は小さくなりますか？

はい。画像のピクセル寸法を縮小すると、画像に含まれるデータ量が劇的に減少するため、画質を損なうことなく最終的なファイルサイズが大幅に軽くなります。

なぜ縦横比（アスペクト比）が重要なのですか？

縦横比とは、画像の幅と高さの比率です。高さを自動調整せずに幅だけを固定値に変更すると、画像が縦長または横長に歪んでしまいます。比率を維持すると、自動的に適切な高さが計算されます。

どのようなフォーマットに対応していますか？

JPG、JPEG、PNG、WebP 形式の画像をアップロードできます。保存時には、元の形式を維持するか、他のサポートされている形式に変換して保存できます。

このサイトで個人写真のサイズを変更しても安全ですか？

非常に安全です。従来のクラウドツールとは異なり、Convertify は現代の HTML5 Canvas API を使用して、すべての処理を100%ローカル（お使いのブラウザ内）で実行します。写真が外部サーバーに送信または保存されることはありません。
